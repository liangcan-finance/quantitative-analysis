# 工作流程

以周为单位，通过五大模块的独立评分（每个模块 0-10 分），加权汇总得到综合得分，再映射到仓位建议区间。**核心是量化打分 → 加权汇总 → 仓位映射**。

数据来源：知识星球（思维纪要社）、专题/投研报告以及个股数据（优先从ifind-finance-data和mx-xuangu这两个skill来获取）、宏观数据（国家统计局、海关总署、央行）、互联网搜索（baidu-search skill）。

## 评分总览

| 模块 | 权重 | 评分范围 | 评分含义（10分=最利好权益） |
|:---|:---:|:---:|:---|
| 一、经济周期与政策 | 25% | 0-10 | 基本面驱动方向 |
| 二、市场情绪与资金面 | 25% | 0-10 | 短期风险偏好 |
| 三、市场估值 | 20% | 0-10 | 安全边际水位 |
| 四、技术分析（缠论） | 15% | 0-10 | 走势结构位置 |
| 五、历史相似环境 | 15% | -2~+2 | 置信度调整 |

**综合得分公式**：`S = Σ(模块评分 × 权重) + 历史环境调整值`

**仓位映射规则**：

| 综合得分区间 | 建议仓位 | 操作方向 | 说明 |
|:---:|:---:|:---:|:---|
| ≥ 8.5 | 满仓（90%-100%） | 积极加仓 | 基本面+情绪+估值共振 |
| 7.0 ~ 8.4 | 偏重（70%-90%） | 逢低加仓 | 多数信号正面 |
| 5.5 ~ 6.9 | 中性（50%-70%） | 均衡持有 | 信号分歧，观望为主 |
| 4.0 ~ 5.4 | 偏轻（30%-50%） | 逢高减仓 | 风险信号增多 |
| < 4.0 | 轻仓（0%-30%） | 主动减仓/对冲 | 系统性风险 |

> 仓位区间是指导性框架，具体幅度需结合核心矛盾判断进行 ±5%~10% 的灵活调整。

# 一、经济周期与政策（权重25%）

这是基本面驱动层，确定市场是"盈利驱动"还是"流动性/情绪驱动"。

## 1.1 经济增长与企业利润

### 指标及数据源
- **规模以上工业增加值**（月度，国家统计局）
- **PMI**（制造业 & 非制造业，国家统计局，每月末公布）
- **工业企业利润总额增速**（月度，国家统计局）
- **上市公司整体归母净利润增速**（季报，Wind 全 A）
- **出口金额同比增速**（月度，海关总署）— 代表外需

### 评分规则

| 条件 | 得分 |
|:---|:---:|
| PMI ≥ 50 且 工业增加值 ≥ 5% 且 企业利润正增长 | 8-10 |
| PMI ≥ 50，但利润增速下滑或出口转弱 | 5-7 |
| PMI 在 49-50 之间震荡，企稳信号不明显 | 3-5 |
| PMI < 49 连续两个月，利润负增长，出口负增 | 0-3 |
| 出现数据真空期或异常波动 | 取最近三个月均值映射 |

### 需要识别的情形
- **阶段性数据扰动**：春节错位、基数效应、极端天气 → 看三个月均值平滑，不因单月波动调整评分
- **结构分化**：名义GDP增速低但新质生产力相关行业高景气 → 评分需要反映结构性方向，而非总量

## 1.2 通胀情况

### 指标
- **CPI、核心CPI**（月度，国家统计局）
- **PPI**（月度，国家统计局）
- **CPI-PPI 剪刀差** → 判断利润在上中下游之间的分配

### 评分规则

| 情形 | 对权益影响 | 得分调整 |
|:---|:---:|:---:|
| CPI 温和（0-2%），PPI 从底部回升 | 利好 | 不调整（或 +0.5） |
| CPI > 3%（通胀高），央行可能收紧 | 利空 | 所在档位 -1 |
| PPI 持续为负（通缩），企业利润率承压 | 利空 | 所在档位 -1 到 -2 |
| CPI 负增长（通缩风险），但政策宽松预期增强 | 偏多 | 所在档位 +0.5 |

## 1.3 宏观经济政策基调

### 分析维度
- **财政政策**：赤字率、专项债发行节奏、特别国债、减税降费力度
- **货币政策**：LPR/MLF利率变动、存款准备金率、公开市场操作力度、社融/M2增速
- **产业政策**：新质生产力方向的具体政策（设备更新、以旧换新、AI、半导体、新能源）
- **房地产政策**：限购松绑、首付比例、房贷利率、保交楼进展

### 评分规则

| 政策组合 | 得分区间 |
|:---|:---:|
| 财政+货币双宽，产业政策密集出台 | 8-10 |
| 宽松但克制（"精准有力"、"托而不举"） | 5-7 |
| 观望期，政策空窗或等待落地 | 3-5 |
| 收紧信号（防过热、去杠杆、加息/提准） | 0-3 |

## 1.4 内需

### 指标及数据源
- **房地产投资/销售额同比**（月度，国家统计局/克而瑞）
- **固定资产投资增速**（制造业/基建/房地产，月度，国家统计局）
- **社会消费品零售总额同比**（月度，国家统计局）

### 评分规则

将三个指标的当月同比增速分别映射，取**最差的那个**打分（短板原则——内需只要有一个引擎熄火，整体就拉不动）：

| 条件 | 得分调整 |
|:---|:---:|
| 三个指标均正增长（社零≥3%、固投≥3%、房地产销售≥0%） | +1 ～ +2 |
| 两个指标正增长，一个轻微负增长（-2%~0%） | 0 ～ +1 |
| 一个指标正增长，两个负增长；或三个均在 0% 附近震荡 | -1 ～ 0 |
| 两个及以上指标明显负增长（社零<0%、固投<0%、房地产销售<-5%） | -1 ～ -2 |

### 分项判断指引

**消费（社零）**：
- 社零同比 > 5% → 消费强劲，内需驱动有力
- 社零同比 3%-5% → 温和复苏
- 社零同比 0%-3% → 偏弱，需求不足
- 社零同比 < 0% → 严重萎缩

> 需剔除汽车/油价等短期扰动，看商品零售（除汽车）和餐饮收入分开判断。同时关注 CPI 食品项和核心 CPI 走势交叉验证消费冷热。

**投资（固投）**：
- 总投资同比 > 5% → 投资驱动强劲
- 制造业投资、基建投资、房地产投资需**分开看**：
  - 制造业投资 > 10% → 产业升级动能强，加分
  - 基建投资 > 8% → 政策发力明显，加分
  - **房地产投资持续负增长** → 内需最大拖累项，-0.5~-1 调整

**房地产销售**：
- 销售面积同比 > 0% → 市场企稳/回暖
- -5% ~ 0% → 筑底中，未确认拐点
- < -5% → 继续下滑，内需信心承压

> 房地产是内需的"领先指标"和"信心指标"，销售端若持续萎缩，后续会传导至投资、就业、消费。房地产对内需拖累越大，评分调整应越偏负。


## 1.5 出口与外部环境

### 指标
- **出口同比增速**（海关总署，月度）
- **美国/欧洲 PMI**（ISM / HCOB，月度）
- **中美关系/关税动态**（新闻事件，跟踪关键时间点）
- **全球贸易景气指数**

### 核心判断
- 外需强（出口双位数增长）+ 汇率稳定 → 加分
- 外需走弱 + 关税升级 → 减分（尤其对出口敏感型行业）
- 人民币汇率大幅贬值（贬破关键心理位）→ 资本外流压力，减分

## 1.6 经济周期评分汇总

```
经济周期总分 = 增长评分 × 0.40 + 政策评分 × 0.25 + 内需调整 + 通胀调整 + 出口调整
```

> 增长评分取 1.1 中各项指标最差的两个的均值（短板原则），政策评分取 1.3 的综合评估。内需/通胀/出口调整各自范围为 -2 ~ +2，三项合计调整范围 -6 ~ +6。最终评分将作为权重 25% 输入到综合得分。

> ⚠️ **增长与内需的关系**：1.1 增长评分反映总量景气（PMI+工业+企业利润），1.4 内需调整反映内需侧结构健康度。两者独立打分是因为：总量景气可能由出口或政策短期拉动，掩盖内需疲弱的真相（如 2026 年 5 月出口 +19% 但社零仅 +0.2%）。扣分项叠加才能真实反映内生动力不足。

# 二、市场情绪与资金面（权重25%）

分别刻画过去**一周**和过去**一个月**的市场热度和资金结构，以**月度数据为主**计算评分（降低周度噪音干扰），周度数据作为边际变化信号进行微调。

## 2.1 市场热度综合指标（1-10分）

### 数据收集原则

所有指标需同时收集**周度**和**月度**两个维度的数据：
- **周度数据**：过去5个交易日的日均值，用于捕捉短期情绪波动和拐点信号
- **月度数据**：过去22个交易日的均值/累计值，用于判定情绪趋势方向，排除短期噪音
- **最终评分以月度数据为主**：月度得分作为基础分，周度边际变化作为 ±0.5~1 的微调

### 基础指标（需收集并归一化）

| 指标 | 数据源 | 周度/月度 | 极端看多信号 | 极端看空信号 |
|:---|:---|:---|:---|:---|
| 涨跌家数比 | 东方财富/同花顺 | 周均 / 月均 | > 2.0（过热） | < 0.5（恐慌） |
| 涨停家数 vs 跌停家数 | 东方财富/同花顺 | 周均 / 月均 | 涨停 > 跌停 3倍以上 | 跌停 > 涨停 |
| 创新高股票数 / 全市场 | 同花顺/Wind | 周均 / 月均 | > 10% | < 1% |
| 创新低股票数 / 全市场 | 同花顺/Wind | 周均 / 月均 | < 1% | > 10% |
| 全市场日均成交额 | 交易所 | 周均 / 月均 | > 1.5万亿（过热） | < 0.6万亿（低迷） |
| 两融余额变动 | 交易所/Wind | 周度 / 月度 | 增 > 3% | 降 > 3% |

### 评分映射（以月度数据为准）

| 月度综合判断 | 月度基础得分 |
|:---|:---:|
| 月度涨多跌少，成交月均 > 1.2万亿，赚钱效应持续扩散 | 8-10（⚠️ 过热警惕） |
| 月度涨跌交替，成交月均 0.8-1.2万亿，赚钱效应尚可 | 6-8 |
| 月度横盘震荡，成交月均 0.6-0.8万亿，方向不明确 | 4-6 |
| 月度跌多涨少，成交月均 < 0.6万亿，亏钱效应扩散 | 2-4 |
| 月度连续大跌，恐慌情绪蔓延，成交缩量至冰点 | 1-2 |

### 周度边际微调

在月度基础得分之上，用周度数据的边际变化进行微调：

| 周度边际变化 | 调整幅度 |
|:---|:---:|
| 周度信号明显改善（如从恐慌→回暖，或周涨家数占比 > 60%）| +0.5 ~ +1 |
| 周度信号与月度方向一致且强化 | +0.5 |
| 周度信号无明显边际变化 | 0 |
| 周度信号明显恶化（如从回暖→再次恐慌，或周跌家数占比 > 60%）| -0.5 ~ -1 |
| 周度信号方向与月度相反（如月度偏多但周度出现极端亢奋）| -1（警惕过热信号） |

> ⚠️ **极度亢奋（月度得分≥8分）要警惕**：此时虽是情绪高点，但往往意味着短期顶部区域。若月度得分 ≥ 8 且周度出现缩量滞涨，应视为减仓预警而非加仓信号。

## 2.2 不同资金动向分析

### 数据维度

每类资金同时关注**周度边际变化**（短期）和**月度趋势**（中期），以**月度累计方向**作为资金面得分的基础判断：
- 如北向资金：周度净买入/卖出看边际变化，月度累计净买入/卖出看趋势方向
- 如两融余额：周度变动看短期情绪，月度变动看杠杆资金信心

### 需要跟踪的资金类别和获取方式

| 资金类型 | 代理指标 | 周度信号 | 月度趋势 | 数据获取 |
|:---|:---|:---|:---|:---|
| **外资（北向）** | 北向资金净买入/卖出 | 单周流入/流出方向 | 近22个交易日累计 | 港交所每日披露 → 东方财富/同花顺汇总 |
| **公募基金** | 偏股基金发行份额、仓位估算 | 发行是否放量 | 月度发行趋势、仓位变化方向 | Wind、星球（思维纪要社渠道调研） |
| **散户** | 新增开户数、银证转账、两融余额 | 两融周变动 | 月度开户趋势、两融余额方向 | 中国结算、交易所、星球 |
| **保险/社保** | 险资权益配置比例 | —（季度数据） | 季度配置方向变化 | 银保监会季度数据 |
| **游资/量化** | 龙虎榜上榜频率、热点换手率 | 周度活跃度 | 月度风格偏好变化 | 东方财富龙虎榜、星球渠道调研 |
| **产业资本** | 增减持公告、回购计划 | 周度增减持净额 | 月度累计增减持方向 | 交易所公告、Wind |

### 综合判断（以月度趋势为主）

| 月度资金面状态 | 得分区间 |
|:---|:---:|
| **多路资金共振流入**（北向月净买入+基金发行回暖+两融上升+产业资本回购>减持） | 7-9 |
| **部分资金流入，部分中性**（北向净买入但基金发行平淡，两融稳定） | 5-7 |
| **存量博弈**（北向、两融、产业资本均无显著方向，结构轮动） | 4-5 |
| **部分资金流出**（北向月净卖出、两融下降、基金发行低迷之一） | 2-4 |
| **多路资金共振流出**（北向大额净卖出+两融持续下降+公募降仓+产业资本净减持） | 0-2 |

### 周度边际微调

在月度资金面基础得分之上，用周度边际变化微调：

| 周度边际变化 | 调整幅度 |
|:---|:---:|
| 周度资金方向较上月明显改善（如北向由净流出转为净流入） | +0.5 ~ +1 |
| 周度与月度方向一致 | 0 |
| 周度方向与月度相反（如月度流入但周度出现大额流出）| -0.5（警惕短期拐点） |
| 周度资金方向较上月明显恶化 | -0.5 ~ -1 |

### 辅助判断要点
- **产业资本大额回购/增持**：底部区域的重要信号（历史经验：增持高峰出现在市场底部附近）
- **散户开户高峰**：往往是市场情绪末端指标，散户大量入场出现在顶部区域附近

## 2.3 情绪评分汇总

### 评分步骤

**Step 1：计算月度热度基础分**
- 根据 2.1 的月度数据，对照月度评分映射表，得出月度热度基础分（1-10分）

**Step 2：计算月度资金面基础分**
- 根据 2.2 的月度资金趋势，对照月度资金面评分表，得出月度资金面基础分（0-9分）

**Step 3：周度微调**
- 热度侧：根据 2.1 的周度边际微调表，对月度热度基础分 ±0.5~1
- 资金侧：根据 2.2 的周度边际微调表，对月度资金面基础分 ±0.5~1

**Step 4：加权汇总**

```
月度热度基础分（周度微调后）= 月度热度基础分 + 周度热度微调
月度资金面基础分（周度微调后）= 月度资金面基础分 + 周度资金微调

情绪评分 = 月度热度基础分（微调后）× 0.60 + 月度资金面基础分（微调后）× 0.40
```

> 热度得分超过 8 时，资金面权重降至 35%（原 40%），热度权重升至 65%（原 60%），防范极端情绪下资金面信号失真。

# 三、市场估值（权重20%）

以 2016 年 1 月为数据起点，对主流指数进行估值水位判断。

## 3.1 核心指数估值水位

### 需要收集的数据

| 指数 | PE（TTM） | PB | PE历史分位数 | PB历史分位数 | 数据源 |
|:---|:---:|:---:|:---:|:---|:---|
| 沪深300 | — | — | — | — | Wind/同花顺/东方财富 |
| 创业板指 | — | — | — | — | 同上 |
| 上证指数 | — | — | — | — | 同上 |
| 深证成指 | — | — | — | — | 同上 |
| 恒生科技 | — | — | — | — | Wind/港交所 |
| 科创50 | — | — | — | — | Wind/上交所 |
| 标普500 | — | — | — | — | Wind/Yahoo Finance |

### 评分规则

计算 **A 股主要宽基指数的 PE 分位数均值**（沪深300+创业板+上证+深成+科创50 的简单平均）：

| PE 分位数均值 | 对权益 | 得分 |
|:---:|:---:|:---:|
| < 20% 分位（显著低估） | 非常有利 | 9-10 |
| 20%-40% 分位（偏低） | 有利 | 7-8 |
| 40%-60% 分位（中枢附近） | 中性 | 5-6 |
| 60%-80% 分位（偏高） | 谨慎 | 3-4 |
| > 80% 分位（显著高估） | 风险 | 1-2 |

> **⚠️ 估值低 ≠ 立刻涨**：低估值可能持续低迷（如 2012-2014 年的 A 股），需结合经济周期判断是"价值陷阱"还是"黄金坑"。
>
> **⚠️ 估值高 ≠ 立刻跌**：高估值在产业趋势向上时可能持续扩张（如 2020 年的消费/医药），需结合基本面趋势判断是"泡沫"还是"合理溢价"。

## 3.2 股债性价比（ERP — Equity Risk Premium）

### 公式
```
ERP = 沪深300股息率 - 10年期国债收益率
```

### 历史比较 & 评分

| ERP 相对于过去5年均值 | 含义 | 得分调整 |
|:---|:---|:---:|
| 高于均值 +1 个标准差以上 | 股票相对债券极具性价比 | 基础分 +2 |
| 高于均值 0~1 标准差 | 有性价比，但并非极端 | 基础分 +1 |
| 在均值附近 ±0.5 标准差 | 中性 | 基础分不变 |
| 低于均值 0~1 标准差 | 债券相对更有吸引力 | 基础分 -1 |
| 低于均值 -1 个标准差以下 | 股票性价比极差 | 基础分 -2 |

## 3.3 估值评分汇总

```
估值评分 = (指数PE分位数得分 × 0.70 + 股债调整值) 
```

- 股债调整值范围在 -2 到 +2 之间
- 最终得分限制在 0-10 分区间内

# 四、技术分析 — 缠论走势研判（权重15%）

运用缠论分析框架，对主流指数（上证指数、沪深300、创业板指）进行走势结构定位。**这一部分需要建立在对缠论基本原理的熟练掌握之上，按以下步骤执行。**

## 4.1 缠论基础操作框架

### 必要前置知识
缠论的核心分析单位是按 **分型 → 笔 → 线段 → 中枢 → 走势类型** 的递归构建过程。在开始分析前，需要先掌握以下概念：
- **顶底分型**：相邻三根K线的不包含关系组合，中间K线最高/最低
- **笔**：顶底分型之间至少5根K线（不含），构成一笔
- **线段**：至少三笔重叠构成一个线段
- **中枢**：连续三段次级别走势重叠的价格区间
- **走势类型**：盘整（一个中枢）、趋势（两个以上同向中枢）

### 操作步骤

**Step 1：K线包含关系处理**
- 对原始K线序列，按时间顺序进行包含关系处理（上升趋势中取高高，下降趋势中取低低）
- 输出"标准K线"序列

**Step 2：寻找底分型和顶分型**
- 相邻三根标准K线，若中间K线最低（底分型）或最高（顶分型）
- 标注所有顶底分型的位置

**Step 3：画笔**
- 顶底分型间隔≥5根标准K线（不含分型本身），上升笔：底到顶；下降笔：顶到底
- 确认笔的连通性：相邻顶底分型不能共用K线
- 输出笔序列

**Step 4：画线段**
- 三笔重叠构成一段
- 标注线段的方向（上升段/下降段）
- 确认当前处于哪个线段中

**Step 5：识别中枢**
- 找出连续三段次级别（笔级别）走势的重叠区间
- 标记中枢的上下沿（ZG/ZD）
- 确认当前走势是在中枢上方、内部还是下方

**Step 6：走势分类**
- 确认当前走势是盘整还是趋势
- 如果是趋势，是三买还是三卖位置
- 判断背驰情况：进入段 vs 离开段

**Step 7：当前位置打分**

| 走势位置 | 含义 | 得分 |
|:---|:---|:---:|
| 下跌趋势末端，底背驰确认后，一买位置 | 中期底部 | 8-10 |
| 二买/三买位置，中枢完成向上突破 | 上攻中继 | 7-8 |
| 中枢内部震荡（方向不明确） | 震荡整理 | 4-6 |
| 上涨趋势末端，顶背驰/一卖位置 | 中期顶部 | 3-5 |
| 二卖/三卖位置，跌破中枢 | 下跌中继 | 1-3 |
| 无清晰结构（无中枢/笔不够） | 无法判断 | 5（中性） |

### 必须包含的输出
- 上证指数当前的中枢区间（ZG/ZD具体点位）
- 当前在第几个中枢内运行
- 进入段和离开段的力度对比（MACD面积或斜率）
- 关键支撑/阻力位（中枢上下沿、重要分型点位）

## 4.2 成交量验证

缠论中强调"量价配合"：突破中枢需要放量确认，背驰段缩量为买点信号。
- 突破中枢时的成交量是否 > 5日均量的 1.5 倍 → 突破有效
- 底背驰段成交量是否萎缩至 5日均量的 60% 以下 → 背驰有效
- 顶背驰段成交量是否创近期天量 → 背驰确认

## 4.3 缠论评分汇总

```
缠论评分 = 走势位置得分 × 0.70 + 成交量验证调整（±0~2）
```

- 成交量验证：放量突破有效 +1，缩量底背驰确认 +0.5，缩量反弹顶部背驰 -1，巨量高位滞涨 -2

## 6.2 核心矛盾判断

评分卡给出量化框架，但最终加减仓建议需要定性判断当前市场最核心的矛盾是什么：

### 三种核心矛盾类型

| 核心矛盾 | 判断依据 | 操作原则 |
|:---|:---|:---|
| **经济基本面驱动** | PMI持续扩张/收缩，企业利润趋势明确 | 顺周期操作：基本面向上加仓，向下减仓。趋势力度最重要，忽略短期情绪扰动。 |
| **政策预期驱动** | 经济数据疲弱但政策频出，市场走预期 | 政策预期驱动的行情：一是判断政策可信度（历史兑现率），二是看预期差大小。通常在政策落地前做多，落地后防"利好出尽"。 |
| **流动性与风险偏好驱动** | 基本面平淡，但资金面宽松、情绪高涨 | 情绪驱动的行情：顺势但警惕拐点。当热度分 > 8 且增量资金耗尽（基金仓位已高、散户开户高峰已过），应考虑减仓。 |

### 核心矛盾处理框架

```
如果核心矛盾是【基本面】→ 经济周期评分权重上升至35%，情绪权重降至20%
如果核心矛盾是【政策】→ 关注政策兑现窗口期，在兑现前维持仓位，兑现后降低预期
如果核心矛盾是【情绪】→ 极度亢奋（情绪分>8）时从严，极度恐慌（情绪分<3）时从宽
```

### 外围带动的变量（全球定价品种）

对于全球定价的金属、原油、科技等受海外影响的板块，需额外关注：
- 美联储利率路径和美元指数走势
- 全球 PMI 及制造业库存周期
- 中美关系事件节点（关税、制裁、技术限制升级）

> 如果核心矛盾来自海外变量（如美联储政策、地缘冲突），则需要单独评估外部冲击对国内仓位的影响，并在报告中给出"海外风险溢价"的明确标注。

## 6.3 加减仓操作建议

### 基于综合得分的仓位指引（重复6.1前的映射规则）

| 综合得分区间 | 建议仓位 | 操作方向 | 具体操作 |
|:---:|:---:|:---:|:---|
| ≥ 8.5 | 90%-100% | 积极加仓 | 若仓位低于80%，可加仓至满仓；关注最后5%资金的"绝对确定性"买点 |
| 7.0 ~ 8.4 | 70%-90% | 逢低加仓 | 在指数回踩关键支撑位时加仓；不加仓追高 |
| 5.5 ~ 6.9 | 50%-70% | 维持观望 | 保持现有仓位，不做大幅调整；可做结构性再平衡 |
| 4.0 ~ 5.4 | 30%-50% | 逢高减仓 | 在反弹中逐步降低仓位；优先减仓浮盈标的 |
| < 4.0 | 0%-30% | 主动减仓 | 减至最低必要仓位；考虑用指数ETF做对冲 |

### 操作节奏
- **建议的频率**：周度回顾，月度重评
- **单次最大调整幅度**：不超过仓位的 20%（避免追高杀跌）
- **连续同向调整的限制**：连续3次减仓后至少停1周观察，避免恐慌性减仓
- **突破性信号处理**：出现突发重大事件（黑天鹅），立即暂停周度框架，转入应急评估

## 6.4 逆向与趋势的平衡

在最终建议中给出"逆向"和"趋势"的双重视角：

| 情绪状态 | 趋势视角 | 逆向视角 | 操作建议 |
|:---|:---|:---|:---|
| 极度恐慌（情绪分≤2） | 趋势向下，空仓观望 | 历史极度恐慌后3/6个月大概率反弹 | 从轻仓（0-30%）转向逢低建仓（30-50%） |
| 极度亢奋（情绪分≥9） | 趋势向上，满仓持有 | 过热后1-3个月回调风险增加 | 从满仓（90-100%）转向适度减仓（70-80%） |
| 中性偏弱（情绪分3-5） | 关注经济周期是否见底 | 低估值+政策底，可逆向布局 | 逐步逢低加仓至中性仓位 |
| 中性偏强（情绪分6-8） | 顺势持有或适度加仓 | 不主动逆向操作 | 维持或适度加仓 |
