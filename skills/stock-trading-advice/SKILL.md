# 经济周期和政策
这是经济的基本面，来确定市场是基本面驱动还是市场资金、风险偏好驱动
-经济增速和企业利润
-出口增速代表的外需
-国内需求包括：社会品零售数据、房地产销售和国内投资
-通胀情况
-当前宏观经济政策基调：刺激经济、托而不举、防止过热

来源
-web
-星球
-专题报告（warrenq）

# 市场情绪和资金风险偏好
对当前市场情绪的刻画，包括市场情绪和资金结构情况分析
-运用综合指标刻画过去一周市场热度：包括涨跌家数比、创新高股票数、创新低股票数等指标。1-10刻画当前热度，10表示市场情绪很高，赚钱效应好，涨家数远高于下跌股票数；1表示市场情绪低；
-不同资金当前动向，包括外资、散户、保险资金、机构资金、游资和量化资金


来源：
-星球
-web

# 当前市场的估值
对当前市场定价进行刻画，数据以2016年1月为起点
-当前主流指数的pe、pb估值水平和估值分位数，包括hs300、创业板、上证、深成和恒生科技
-美股的指数的整体pe估值水平
-hs300股息率相对十年国债的比较


来源：
-星球
-公司报告

# 技术分析
利用缠论刻画主流指数的当前走势，分析未来走势可能性
-学习缠论
-刻画当前市场的位置，短期和中期
-未来走势判断

来源：
-星球
-公司报告

# 历史上的类似环境
类似当前的k线走势形态的历史时间段，分析当时的宏观环境、市场风险偏好
-2016年以来的k线走势，找到类似当前上证指数走势的阶段，列出时间段
-分别列出这些时间段中的宏观环境、市场风险偏好和市场结构特征

来源：
-星球
-公司报告

# 确定加减仓
综合判断给出加减仓的建议，同时考虑逆向和趋势产
-判断当前的核心矛盾：经济基本面？政策预期？还是风险偏好？
-如果是经济基本面驱动的，判断经济增长或收缩的持续性
-如果是政策预期驱动的，判断政策的可靠性
-如果是流动性和风险偏好驱动的，判断当前的市场情绪的持续性
-对于外围带动的产业投资（全球定价的金属、原油和科技），需要特别提示
