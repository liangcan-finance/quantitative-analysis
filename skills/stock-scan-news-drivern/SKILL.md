description :当用户需要对未来事件或着当下事件进行推演、结合当前市场选择投资标的时使用这个skill，返回可以看到跟事件相关的核心标的推荐。

# 事件
需要清晰定义事件，并对事件进行分类，可以是未发生也可以已经发生
-地缘事件（比如俄乌战争）
-政策事件（比如双碳政策）
-重要会议（比如两会、中央经济工作会议）
-监管政策（比如证监会等部委政策）

来源
-web
-星球
-专题报告（warrenq）

# 链接事件和市场投资方向
链接事件跟投资的关系
-未发生的事件：对事件结果展望的主流观点
-已发生的事件：对事件影响的主流观点梳理
-事件如何影响总量：大盘走势、市场结构


来源：
-星球
-专题报告

# 梳理影响的行业和股票池
针对事件影响的行业方向梳理股票池
-影响的行业逻辑梳理（比如俄乌战争对军工、能源、资源行业）
-行业投资股票池


来源：
-星球
-公司报告

# 决定投资池
事件驱动投资的个股选择，筛选和排出条件给定，5-10个的股票池。
排除条件
-st股
-前十大持仓没有任何公募基金

筛选条件
-业绩期前1个月需要考虑业绩预期
-个股标签的走势更强（比如盐湖股份同时有钾肥和锂矿的标签，俄乌战争事件影响钾肥，同时考虑21年电池的景气度）
-大市值股票给更大权重（或者40%大市值）
