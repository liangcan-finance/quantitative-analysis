PI投资-资产选择

在仓位给定的基础上，确定资金在红利、主题、产业和事件驱动投资的比例，并推荐的主题、产业和事件方向
---
name : 资产选择
description :组合仓位择时完成之后，在不同的投资方向上进行资金分配，包括红利、主题、产业和事件驱动四类资产；其中各类主题考虑是市场情况（在有鱼的地方抓鱼），原则上契合市场的方向上投入；
红利驱动投资是指市场资金选择高股息资产，并带动分红资产持续上涨的投资；
主题驱动投资事指在产业仍然孕育期间的投资，渗透率在5%左右，业绩还难以体现在上市公司，但是因为资金或者信息驱动的主题投资；
产业驱动投资是指在产业高速增长阶段的投资，渗透率5%以上并快速提升，业绩开始体现在上市公司，并呈现产业扩散，产业链的供给和需求均高速增长；
事件驱动投资是指对未来某一事件是否发生、发生之后对资产的影响进行的投资
---

# 当前市场的判断

当前市场的投资风格主线
-不同时间段市场的投资主线不同，可以定义当前三个月和半年的投资属于什么投资风格（比如2023年红利驱动投资；2026年上半年可能是产业驱动投资；2025年是主题驱动和产业驱动投资）
-当前投资风格主线的宏观、政策和产业背景
-当前的投资风格主线中，主要的主题驱动投资和产业驱动投资有哪些

来源
-web
-星球
-专题报告（warrenq）

# 资金分配
确定资金分配
-资金分配原则：基准情景下如果仓位100%，产业驱动投资30%、主题驱动投资30%、红利驱动投资30%、事件驱动投资10%；仓位变化时四类投资同比例变化
-结合当前市场的投资风格主线调整资金分配，匹配当前市场的投资风格主线，增加10%-20%的持仓，同时调整其他持仓

来源：
-星球
-专题报告

# 主题方向选择
-主题驱动投资容量可投市值在500亿以上；
-有政策、会议、业绩及其他可能催化
-走势上呈现抗跌或者稳定资金吸筹的态势

来源：
-星球
-公司报告

# 产业方向选择
产业的筛选
-产业驱动投资容量可投市值在500亿以上
-业绩连续两个季度超预期，兑现高增长
-走市场呈现抗跌后者稳定资金吸筹的态势


来源：
-星球
-公司报告

# 红利资产选择
红利资产筛选
-列出不同产业的股息率，业绩增长预期比较
-业绩5%以上的增长预期，且股息率排名靠前的红利标签资产

来源：
-星球
-公司报告

# 事件选择
事件的筛选
-未来需要关注的地缘、国内国际会议、产业会议等，核心关注中美会议、地缘、中央政治局会议和规划会议
-根据对资本市场重要性大小进行筛选，并排序


来源：
-星球

# 推荐资产选择
确定资产选择，输出
-当前市场判断
-资金分配
-主题驱动投资选择的方向（按重要性列举10个），简要逻辑
-产业驱动投资选择的方向（按重要性列举10个），简要逻辑
-事件驱动投资选择的方向，按重要性列举，简要逻辑
-红利驱动投资选择的方向，直接输出选择标的的股息率和业绩增长
