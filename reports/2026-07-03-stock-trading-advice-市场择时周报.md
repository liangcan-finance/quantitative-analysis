## 2️⃣ 核心矛盾判断

### 核心矛盾类型：**流动性与风险偏好驱动**

当前市场并非由基本面全面复苏驱动（PMI仅50.3、社零转负），而是由**天量流动性和AI科技主题风险偏好**推动的结构性行情：

- **K型分化极致**：科创50创历史新高，而消费/地产板块重回924前水平
- **两融余额首破3万亿**：杠杆资金集中涌入AI硬件、半导体等高景气赛道，TMT成交占比达历史极值
- **外需强 vs 内需弱**：出口+19.4% 与 社零-0.6% 形成巨大剪刀差，经济"一条腿走路"

⚠️ **拐点警惕信号**：7月2日沪深300单日暴跌3%+叠加两融天量 → 短期过热后获利了结压力增大。

### 按核心矛盾调整权重

```
情绪驱动 → 极度亢奋（情绪分>8）时从严，当前情绪分5.5未触发极端阈值
但结构性亢奋（科技赛道）已接近极致 → 整体维持中性偏谨慎
```

## 4️⃣ 加减仓建议

### 📌 仓位指引

| 项目 | 建议 |
|:---|:---|
| **综合得分** | **4.08** |
| **建议仓位区间** | **35%-50%** |
| **操作方向** | **逢高减仓**（尤其浮盈科技仓位） |
| **单次最大调整** | 不超过总仓位15% |

### 🎯 具体操作建议

1. **科技/AI仓位**：已累计较大涨幅（科创50创历史新高），建议在反弹中逐步止盈1/3至1/2，锁定利润
2. **防御配置**：可适度增加红利/高股息仓位（10Y国债仅1.74%，股息率>4%的红利资产极具相对价值）
3. **观望方向**：消费/地产链不宜抄底，需等待社零数据拐点确认
4. **加仓条件**：若沪深300回踩4500-4600区间+缩量企稳，可加仓至中性（50-60%）

### ⚠️ 海外风险溢价

- 美联储偏鹰（加息预期升温）→ 对全球科技股估值形成压制
- 美伊冲突扰动 → 油价不确定性
- 中美关系 → 关税/技术限制需持续关注
